# Psynovia Website

Statisches Website-Paket für Psynovia.

## Struktur

- `index.html` – Onepager: Hero → Erklärung → Ablauf → Pricing → Footer
- `datenschutz.html` – Datenschutzseite aus Projektdateien
- `impressum.html` – Impressum aus Projektdateien
- `wartung.html` – Wartungs-/Aufbauseite
- `components/` – unveränderte Original-Komponenten aus den Projektdateien
- `assets/images/` – bereitgestellte Bilddateien
- `styles/onepager.css` – verbindende Onepager-/Navigationslogik
- `scripts/onepager.js` – Burger-Menü, Smooth Scroll, iFrame-Autoheight

## Deployment

Direkt auf Netlify deploybar. Kein Build-Step notwendig.
